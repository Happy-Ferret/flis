flis is a Wayland tiling compositor inspired by [sway][sway] and [i3][i3],
written in [Go][golang].

SUPER MEGA ALPHA!!!

Don't expect anything to work... yet!

### Dependencies

* [wlc][wlc]
  * [go-wlc][go-wlc]
  * [go-xkbcommon][go-xkbcommon]
* wayland
